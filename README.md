## Installation
1. Clone the Repository
    ```bash
    git clone https://github.com/othneildrew/Best-README-Template.git
    ```
2. Install packages
    - Yarn
        ```bash
        yarn install
        ```
    - NPM
        ```bash
        npm install
        ```

3. Setting `.env` file
    ```env
    PORT = 5000
    MONGODB_URI = local | cloud
    ```
4. And run the API
    ```bash
    yarn start
    # or
    npm start
    ```
